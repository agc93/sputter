What is Sputter?

Sputter is a flexible and configurable API server designed to gather hard drive and SSD temperatures from a variety of sources and make them available through a CLI, HTTP API or MQTT.

speed

Monitoring

Monitor the temperatures of any number of drives through a number of adapters, all available with a simple API.

terminal

CLI or API

Sputter is available as either a simple command-line interface (CLI) for maximum ease of use, or a configurable API server for maximum flexibility.

settings

Configurable

Highly configurable API server supports automatically filtering drives, gathering measurements, and even publishing them over MQTT.
